# github-notes

### Git (version control software)
<p> Git is version control. rather than you having to keep track of separate files and histroy yourself manually, Git is a software that does that for you. It keeps track of all of your history that you're doing on the computer. Its not just only for you, but it allows many people to collaborate on the project together.  </p>

### Github (web service)
<p> Github is a website where you can do projects that have version control on it. You can sign up and have an account to do stuff on a website. It basically runs the Git software behind the scenes.  </p>

### Repo (file)
<p> Repo just stands for repository where you add and have multiple files with it. 
  
### Commit (change)
<p> Commits are changes you make to a file and those commmits allow you to browse history and each one of those commits have a unique identifier for it </p>
